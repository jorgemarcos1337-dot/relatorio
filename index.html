💻 Instruções para o Programador (Funcionalidades Chave)

  • Estado Único (One-Page Core): A página não deve recarregar. Filtros devem atualizar os gráficos e tabelas instantaneamente (via JavaScript).
  • Cross-Filtering Total: Esta é a funcionalidade mais importante. Clicar em um elemento de qualquer gráfico (ex: a fatia "Desistentes" da pizza, a barra "712N", a faixa etária "5 anos") deve filtrar TODOS OS OUTROS GRÁFICOS E TABELAS na página inteira.
  • Tooltips Detalhados: Ao passar o mouse sobre qualquer elemento de um gráfico (barras, fatias, pontos de linha), uma janela flutuante (tooltip) deve exibir os números exatos e porcentagens correspondentes.
  • Exportação de Dados: O botão "Exportar para CSV" no cabeçalho deve gerar um arquivo com os dados da visão atual, respeitando os filtros aplicados.
  • Tabelas Ordenáveis: As colunas das tabelas detalhadas devem ser ordenáveis ao clicar no cabeçalho.
  • Layout Responsivo: Garantir que o design se adapte perfeitamente a desktops, tablets e celulares, conforme as regras de CSS já estabelecidas.
